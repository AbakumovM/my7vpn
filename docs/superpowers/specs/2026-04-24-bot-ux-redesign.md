# Спецификация: Редизайн UX бота

**Дата:** 2026-04-24
**Статус:** утверждён

---

## Контекст

Бот изначально строился вокруг концепции «устройства» — пользователь создавал отдельные устройства с независимыми подписками. После миграции на Remnawave модель изменилась: **один пользователь = одна подписка** с лимитом устройств (1/2/3) и одной ссылкой для happ. Текущий UX не отражает эту модель.

### Проблемы текущего UX

- «Мои устройства» — устаревшая концепция, у пользователя одна подписка
- «☠️ Не работает VPN» — агрессивная формулировка на главном экране
- Инструкции ссылаются на v2RayTun, сейчас используется happ
- Шаг «выбор типа устройства» (Android/iOS/Windows) не нужен — happ одинаковый для всех платформ
- Реферальный экран без статистики, кнопки не по теме
- Нет кнопки личного кабинета (сайт)
- Нет кнопки «Пригласить друга» на главном экране

## Главное меню (/start)

### Пользователь с активной подпиской

**Текст сообщения:**
```
👋 Привет, {имя}!

✅ Подписка активна до {дата}
📱 Устройств: {используется} / {лимит}
💰 Баланс: {баланс}₽
```

**Кнопки (сетка 2×2 + 2×1):**

| Левая | Правая |
|-------|--------|
| 📋 Подписка | 🔄 Продлить |
| 📖 Инструкция | 👫 Друзья |
| 🌐 Кабинет | 💬 Поддержка |

- **🌐 Кабинет** — URL-кнопка, открывает сайт (личный кабинет для оплаты вне Telegram)
- **💬 Поддержка** — URL-кнопка, открывает чат `t.me/{admin_username}`

### Новый пользователь (без подписки)

**Текст сообщения:**
```
👋 Привет, {имя}!

У вас пока нет активной подписки.
Подключите VPN и защитите свои данные! 🚀
```

**Кнопки:**

| Левая | Правая |
|-------|--------|
| 🚀 Подключить VPN (на всю ширину) | |
| 📖 Инструкция | 👫 Друзья |
| 🌐 Кабинет | 💬 Поддержка |

- **🚀 Подключить VPN** — занимает всю ширину (grid-column: 1/3), выделена цветом
- Запускает тот же флоу покупки что и «🔄 Продлить»

## Экран «📋 Подписка»

Единый экран вместо списка устройств.

### Активная подписка

**Текст:**
```
🔐 Ваша подписка

📅 Активна до: {дата}
📱 Устройств: {используется} / {лимит}
💳 Последний платёж: {сумма}₽

🔗 Ваш ключ для happ:
{subscription_url}
```

**Кнопки (2×1 + 1×1):**

| Левая | Правая |
|-------|--------|
| 🔄 Продлить | 📖 Инструкция |
| 🏠 Главное меню (на всю ширину) | |

### Подписка истекает (≤ 7 дней)

Дата подсвечена предупреждением:
```
📅 Активна до: {дата} (N дней!)
⚠️ Продлите подписку, чтобы VPN не отключился
```

Кнопка «🔄 Продлить подписку» — на всю ширину, выделена цветом.

### Нет подписки

```
У вас пока нет активной подписки.
```

Кнопка «🚀 Подключить VPN» — на всю ширину, выделена цветом.

## Флоу покупки / продления

**3 шага** (было 4 — убран выбор типа устройства, т.к. happ универсальный).

### Шаг 1: Количество устройств

**Текст:**
```
📱 Сколько устройств подключить?

Одна подписка — на все ваши устройства: телефон, компьютер, телевизор.
```

**Кнопки (столбец):**
- `📱 1 устройство` — обычная
- `📱📱 2 устройства` — обычная
- `📱📱📱 3 устройства` — выделена зелёным, бейдж `⭐ хит`

### Шаг 2: Срок подписки

**Текст:**
```
⏳ Выберите срок подписки

📱📱📱 {N} устройств(а)
```

**Кнопки (столбец, цены из `TARIFF_MATRIX`):**
- `1 мес — {цена}₽` — обычная
- `3 мес — {цена}₽ -14%` — выделена зелёным, бейдж `⭐ хит`
- `6 мес — {цена}₽ -29%` — выделена синим, бейдж `💰 выгодно`
- `12 мес — {цена}₽ -38%` — обычная

### Шаг 3: Подтверждение

**Текст:**
```
🔐 Подтвердите подписку

📱 Устройств: {N}
⏳ Срок: {M} месяцев
💰 Стоимость: {цена}₽
🎁 Бонус: -{бонус}₽

💳 К оплате: {итого}₽
```

**Кнопки (сетка 1×2):**

| Левая | Правая |
|-------|--------|
| ✅ Оплатить | ❌ Отмена |

После «✅ Оплатить»:
- `yookassa.enabled = true` → ссылка для оплаты (автоподтверждение через webhook)
- `yookassa.enabled = false` → QR-код + «Я оплатил» (ручное подтверждение админом)

## Экран «📖 Инструкция»

### Шаг 1: Выбор платформы

**Текст:**
```
📖 Инструкция по подключению

Выберите вашу платформу:
```

**Кнопки (столбец):**
- 📱 Android
- 🍏 iPhone / iPad
- 💻 Windows
- 💻 MacOS
- 📺 Android TV
- 🏠 Главное меню

### Шаг 2: Инструкция для платформы

**Пример (Android):**
```
📱 Подключение на Android

1️⃣ Скачайте happ:
→ Google Play (ссылка)

2️⃣ Откройте приложение

3️⃣ Нажмите ➕ и вставьте ваш ключ подписки

4️⃣ Нажмите «Подключиться»

✅ Готово! VPN активен.
```

**Кнопки:**
- 📋 Мой ключ подписки — отправляет subscription_url для копирования
- ◀️ Выбор платформы — назад к списку
- 🏠 Главное меню

Аналогичные инструкции для iOS, Windows, MacOS, TV — с соответствующими ссылками на happ для каждой платформы.

## Экран «👫 Друзья»

**Текст:**
```
👫 Пригласи друга

Ты получаешь 50₽, друг — 7 дней VPN бесплатно

📊 Твоя статистика:
Приглашено: {count} друга
Заработано: {total}₽
💰 Баланс: {balance}₽
```

**Кнопки (столбец):**
- `📤 Поделиться в Telegram` — выделена зелёным, Telegram switch_inline_query
- `📋 Скопировать ссылку` — бот отправляет ссылку отдельным сообщением для копирования в любой мессенджер
- `🏠 Главное меню`

### Реферальная статистика

Требуется добавить в `UserView` метод `get_referral_stats(telegram_id) -> ReferralStats`:
```python
@dataclass(frozen=True)
class ReferralStats:
    invited_count: int    # сколько друзей пришло по ссылке
    total_earned: int     # сколько всего заработано (invited_count × 50)
    balance: int          # текущий баланс
```

## Убираемые элементы

| Что убираем | Причина |
|-------------|---------|
| Кнопка «☠️ Не работает VPN» | Заменена на «💬 Поддержка» (URL-кнопка → чат с админом) |
| Шаг «тип устройства» (Android/iOS/Windows) в покупке | happ одинаковый для всех платформ |
| Список устройств (экран «Мои устройства») | Одна подписка = один экран |
| Упоминания v2RayTun в инструкциях | Заменены на happ |
| Кнопка «Помощь 🆘» | Заменена на «📖 Инструкция» (более конкретно) |
| `get_keyboard_type_comp()` | Больше не нужна (нет выбора типа устройства) |
| `get_keyboard_tariff_for_update()` | Заменена единым `get_keyboard_tariff()` |
| `ActualTariff` enum | Цены берутся из `TARIFF_MATRIX` |

## Технические заметки

- **URL-кнопки** (`🌐 Кабинет`, `💬 Поддержка`) — `InlineKeyboardButton(text=..., url=...)`, не нужны хендлеры
- **Telegram share** (`📤 Поделиться в Telegram`) — `InlineKeyboardButton(text=..., switch_inline_query=реферальная_ссылка)`
- **Бейджи** (`⭐ хит`, `💰 выгодно`) — часть текста кнопки в Telegram (нельзя стилизовать как в макете, но можно добавить эмодзи: `⭐ 3 мес — 900₽ (-14%) хит`)
- **Процент скидки** — вычисляется от цены за 1 мес: `discount = 1 - (price / (month_price * months))`
- **Subscription URL** на экране подписки — отправляется в `<code>` для удобного копирования или отдельным сообщением с `parse_mode="Markdown"` и обёрткой в backticks
